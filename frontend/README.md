# novel-frontend

Frontend for **MeTruyenChu** — a Vietnamese web novel reading platform with TTS audio.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** TailwindCSS
- **State:** TanStack Query (server) + Zustand (client)
- **Audio:** howler.js + wavesurfer.js
- **PWA:** @ducanh2912/next-pwa

## Getting Started

```bash
npm install
npm run dev
```
